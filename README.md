# 🍕 28 The Kitchen - PWA

A Progressive Web App (PWA) for **28 The Kitchen**, designed for mobile-first ordering.  
Built with **HTML, CSS, JavaScript**.

---

## 🌟 Features
- 📱 Installable on Android/iOS via **Add to Home Screen**
- 🍔 Menu browsing (Pizzas, Burgers, more…)
- 🟢 Send order directly to **WhatsApp**
- ⚡ Works offline with Service Worker caching
- 🎨 Responsive UI with clean design
